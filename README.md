# README

****Initialize Git

* $ git config --global user.name "Your Name"
* $ git config --global user.email "you@youraddress.com"
* $ git config --global push.default matching
* $ git config --global alias.co checkout
* $ git init
					
* Add Files To Git

* git add .
* git commit -am "Commit Message"
* git push
